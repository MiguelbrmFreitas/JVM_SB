## Como compilar

### Linux
Clone o projeto, vá para a pasta e digite o comando
```
make
```
E a saída da compilação estará na pasta build

### Windows
Garantindo que está com o MINGW instalado, clone o projeto e vá até a pasta, digitando o comando
```
mingw32-make
```
